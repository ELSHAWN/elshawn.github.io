The following text has to be placed according to the platform:

Facebook - The song name and ELSHAWN has to be mentioned in the description of the post where my music is used. ELSHAWN has to be hyperlinked with my official Facebook page.

eg - Music: Song Name by ELSHAWN

Twitter - The song name and ELSHAWN has to be mentioned in the tweet where my music is used. ELSHAWN has to be hyperlinked with my official Twitter handle.

eg - Music: Song Name by @IAmELSHAWN

Instagram - The song name and ELSHAWN has to be mentioned in the photo or video post/story where my music is used. ELSHAWN has to be hyperlinked with my official Instagram handle.

eg - Music: Song Name by @iamElshawn

YouTube - The song name and ELSHAWN has to be mentioned in the video description where my music is used.

eg - Song Name by ELSHAWN
https://www.youtube.com/channel/UCVg1qM9i0SZxvWvIj64mroQ

Website/blog - The song name and ELSHAWN have to be mentioned in the caption of the video. If the video featuring my music is on the homepage then the credits can be mentioned in the footer area. ELSHAWN can be linked to my YouTube channel or my official website

eg - Music credits: Song Name by ELSHAWN
